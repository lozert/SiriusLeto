# SciTinder

## Как запустить проект

### 1. Установите Docker Desktop

Перейдите на сайт:

https://www.docker.com/

Скачайте `Docker Desktop` и установите его на компьютер.

После установки запустите Docker Desktop и дождитесь, пока он полностью включится.

### 2. Откройте папку проекта в терминале

Откройте терминал именно в той папке, где находится проект `TestConnetion`.

Например, если проект лежит по этому пути:

```bash
d:\SciTinder\TestConnetion
```

то терминал должен быть открыт именно в этой папке.

### 3. Запустите проект

Введите в терминале команду:

```bash
docker-compose up -d
```

Эта команда запустит проект в контейнерах Docker в фоновом режиме.

Нужно немного подождать, пока все сервисы поднимутся.

### 4. Откройте сайт в браузере

После запуска откройте в браузере ссылку:

http://localhost:5173

### 5. Проверьте работу проекта

Если сайт открылся, значит проект запущен и им уже можно пользоваться и проверять его работу.
